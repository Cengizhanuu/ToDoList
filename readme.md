# To Do List Aplication

## Used

 - Grunt
 - Bower
 - Browserify

## Run Project
```sh
$ npm i
$ bower i
$ grunt // geliştirmeye devam etmek için
$ grunt build // bu komut javascript ve cssleri sıkıştırarak build verir.
```
## Grunt Tasks
  - grunt-sass
  - grunt-contrib-cssmin
  - grunt-contrib-uglify
  - grunt-browserify
  - grunt-contrib-watch
